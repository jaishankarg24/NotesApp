## Cloning and starting application
1. - git clone https://github.com/jaishankarg24/NotesApp.git
1. - cd mynotes-app
2. - npm install
3. - npm run server //STARTS JSON SERVER ON PORT 5000
4. - npm start  //STARTS REACT SERVER
